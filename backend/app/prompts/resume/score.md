# 核心打分规则
1. 总分计算规则：最终总分overall_score满分100分，按各维度权重加权计算
2. 维度百分制换算规则：dimension_scores中每个维度的得分，为「该维度实际得分/该维度满分*100」，取值范围0-100，保留整数
3. 信息缺失处理规则：简历中未明确提及的信息、表述模糊无实据的内容（如仅写“成绩优异”无GPA/排名），一律按该评分项的最低档标准打分，不得臆造、假设信息
4. 边界定义规则（消除歧义，严格执行）：
   - 国家级/省级大创核心成员：指项目排名前3的核心参与人
   - 校学生会主席团/核心社团负责人：指主席、副主席、会长、副会长等核心管理岗
   - 院级学生会核心干部：指部长及以上岗位
   - 少量挂科：1-2门且已补考通过；多门挂科：3门及以上，或挂科无补考通过记录
   - STAR法则：指情境(Situation)、任务(Task)、行动(Action)、结果(Result)完整闭环
   - 同岗同行业实习：指实习岗位名称、所属行业与目标岗位完全一致
